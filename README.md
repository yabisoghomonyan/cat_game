A simple Python game built with Pygame, where the player controls a cat and interacts with different objects in the game world.

This project was created for learning purposes and practicing Python, object-oriented programming, and basic game development concepts.

# Features

Cat character with movement

Interactive objects (threads, fish, cabinets, walls, flowers)

Sound and music support

Modular code structure using classes

# Technologies Used

Python

Pygame

# Project Structure
cat_game/
│
├── main.py              # Main game loop
├── classes_for_game.py  # Game object classes (Cat, Fish, Wall, etc.)
├── function.py          # Helper functions
├── music.py             # Music and sound handling
├── assets/              # Images and sounds (if included)
└── README.md

# How to Run the Game

Clone the repository

git clone https://github.com/yabisoghomonyan/cat_game.git


Install dependencies

pip install pygame


Run the game

python main.py
